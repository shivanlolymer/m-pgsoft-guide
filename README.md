# m-pgsoft-guide

## 项目简介

`m-pgsoft-guide` 是一个用于归档和发布多个独立 HTML 页面的仓库。本仓库不针对任何特定域名或网站，仅作为通用页面集合，便于集中管理和访问。

## 目录结构

```
m-pgsoft-guide/
├── README.md          # 本文件
├── index.html         # 总索引页（可选）
├── pages/             # 存放所有独立 HTML 页面
│   ├── example.html   # 示例页面
│   └── ...
└── assets/            # 公共资源（CSS、JS、图片等）
    ├── css/
    ├── js/
    └── images/
```

- **pages/**：所有归档的 HTML 页面存放于此，每个页面为一个独立文件。
- **assets/**：存放页面所需的公共静态资源，如样式表、脚本和图片。
- **index.html**（可选）：仓库根目录下的索引页，可列出所有页面链接。

## 页面归档说明

- 每个页面均为独立的 HTML 文件，不依赖外部数据库或服务端环境。
- 页面内容仅包含静态 HTML、CSS 和 JavaScript，部分页面可能引用公共资源。
- 归档页面可能包含演示、参考、工具或说明性质的内容，不针对任何具体网站或域名。
- 页面文件命名应具有描述性，建议使用英文小写字母、数字和连字符，如 `example-page.html`。

## 维护说明

- 本仓库由维护者不定期更新，新增或修改页面时，请保持目录结构整洁。
- 新增页面请放入 `pages/` 目录，并在 `index.html`（如有）中添加对应链接。
- 公共资源建议放入 `assets/` 目录，避免在页面中嵌入过多重复样式或脚本。
- 如页面引用外部资源（如 CDN 库），请确保链接稳定可靠，并注明来源。
- 提交更改前请检查页面是否可正常渲染，无明显的样式或功能问题。

## 许可

本仓库内容仅供归档与参考，不提供任何明示或暗示的保证。如有引用第三方资源，请遵守其各自许可协议。
